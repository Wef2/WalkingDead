# Walking Dead?
걷기 운동을 게임과 결합해 다이어트, 재미 일석이조의 효과를 창출

#### 일반 모드 (Walking Mode)

- 걸을 때 마다 1코인 획득
- 걸은 수 차트로 보여주기 (코인 획득 수)
- 코인으로 구매하여 코인 증가률 상승 가능
- 아이템 업그레이드 
  - 방망이 -> 단검 -> 장검
  - 양말 -> 신발 -> 자전거 -> 자동차 -> 비행기 

#### 미션 모드 (Running Mode)

- 출발지와 목적지를 선택
- 좀비가 사용자를 따라오기 시작
- 길목을 지키는 좀비는 스킬(또는 무기)을 이용하여 잡거나, 피해가야함
- 좀비들이 가까워질 수록 사운드가 커짐
- 좀비를 잡는 스킬은 모은 코인으로 구매가능
  - ex. 5단계는 약한 스킬로 깰 수 있음
